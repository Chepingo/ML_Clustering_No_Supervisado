# ML_Clustering_No_Supervisado
Este proyecto aplica un proceso completo de KDD sobre un dataset multivariante de más de 1000 registros y 16 variables, desde la limpieza y el EDA hasta el clustering no supervisado en R. Usa la correlación de Pearson y V de Cramer para hallar relaciones clave, segmentando con K-Means y DBSCAN, y evaluando con silueta, el Davies-Bouldin, ARI y NMI.
